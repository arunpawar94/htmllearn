Advance selectors:Selectors are used to apply css properties on specific elements in html. Youttube Video Link....

There are 4 categories of selectors:
1. Simple selectors
2. Combinators selectors
3. Attribute selector
4. Psedo-classes & Pseudo-elements

1. Simple selectors
There are 4 types of Simple selectors
1. type selectors
Syntax: 
            body
            {
                color:red;    
            }

2. class selector
Syntax: 
            .class_name
            {
                color:red;    
            }

3. id selector
Syntax: 
            #id_name
            {
                color:red;    
            }

4. universal selector
Syntax: 
            *{
                color:red;    
            }
also can be given as:
Syntax: 
            #id_name *
            {
                color:red;    
            }

2. Combinators selectorstwo or more tag or id or class etc to make selectors.
There are 4 types of Combinators selectors
1. Descendant selectors
Syntax: 
            .class_name p
            {
                color:red;    
            } 

            p span
            {
                color:red;    
            } 

            div p span
            {
                color:red;    
            }

2. Direct Child selectors or child selctors (they show effect of css on direct child of parent element and not on the grand child)
Syntax: 
            .class_name or id_name > p
            {
                color:red;    
            }
        sign of greater than i.e. ">" is used

3. Adjacent Sibling selectors (here "+" sign is used. it works only on the just succeding the element)
for example in below syntax property "color:red;" is applied only the ul which is just succed the div but if another ul is also there then the property is not applied there.
Syntax: 
            div + ul
            {
                color:red;    
            }
        sign of plus i.e. "+" is used

4. General Sibling selectors (here "~" sign is used. it apply on all succeding that element)
for example in below syntax property "color:red;" is applied on all the ul which are succed the div.
Syntax: 
            div ~ ul
            {
                color:red;    
            }
        sign of tiddal i.e. "~" is used

3. Attribute selector Youttube Video Link....
There are 7 way to give attribute selector:

1. tag_name[attr_name]
Syntax: 
img[width]
            {
                border:6px solid red;
            }

apply on "img" tag which is using "width" attribute.

2. tag_name[attr_name='val']
Syntax: 
img[width='300px']
            {
                border:6px solid red;
            }

apply only on "img" tag whose width exactly 300px i.e. "width=300px" attribute.

3. tag_name[attr_name^='val']
Syntax: 
img[width^='3']
            {
                border:6px solid red;
            }

apply on "img" tag whose value of width is starting 3.

4. tag_name[attr_name|='val']
Syntax: 
img[title|='image']
            {
                border:6px solid red;
            }

apply on "img" tag whose value of title is 'image-' i.e. "image" and "-" both are present.

5. tag_name[attr_name$='val']
Syntax: 
img[title$='flower']
            {
                border:6px solid red;
            }

apply on "img" tag whose value of title attribute is ending with "flower".

6. tag_name[attr_name*='val']
Syntax: 
img[title*='age']
            {
                border:6px solid red;
            }

apply on "img" tag in which "age" is present anywhere in the value of title.

7. tag_name[attr_name~='val']
Syntax: 
img[title~='image']
            {
                border:6px solid red;
            }

apply on "img" tag in which value of title is "image" and there must be space before and after the word "image".

5. Pseudo-elements "::" is used.
Youtube Video Link....
There are over-all 6 types of psedo-elements:
  1. tag_name::after

    2. Syntax: 
                    p::after
                {
                    content:"hello";
                }
    content is used for before and after.before is for first child and after is used for last child.
  2. tag_name::before
  3. tag_name::first-letter
  4. tag_name::first-line
  5. tag_name::selection
  6. tag_name::placeholder
  7. ::marker

    8. used to change the color of marker of ordered list or unordered list.
    Syntax: 
                    ::marker
                {
                    color:red;
                }
                The above syntax change the marker color of all lists wheather it is ordered or unoredered list.
                
                ul li::marker
                {
                    color:red;
                }
                The above syntax change the marker color of only unoredered list.
  8. ::file-selector-button

    9. use to design the file uploading button which we use in input tag as a type of "file".
    Syntax: 
                    input[type=file]::file-selector-button
                {
                    css property
                }
                
                we can use it with other pseudo class also such as for :hover
                input[type=file]::file-selector-button:hover
                {
                    css property
                }
