text-decoration properties are:

1.text-decoration-line:underline; used to draw line.
The values are:

a) underline Eg. My name is Arun Pawar. Draw line below the text.

b) overline Eg. My name is Arun Pawar. Draw line above the text.

c) line-through Eg. My name is Arun Pawar. Draw the line in the middle of the text.

d) none Eg. My name is Arun Pawar.
value "none" is used to remove any line such as underline fo anchor tag.

2.text-decoration-color:blue;used for color to the text-decoration-line we used.
Eg. My name is Arun Pawar.

3.text-decoration-style:wavy;used for different style to the text-decoration-line we used.
The values are:

a) solid Eg. My name is Arun Pawar.

b) dotted Eg. My name is Arun Pawar.

c) dashed Eg. My name is Arun Pawar.

d) double Eg. My name is Arun Pawar.

e) wavy Eg. My name is Arun Pawar.

Shorthand for text-decoration is:
text-decoration:underline green double; (line color style)Eg. My name is Arun Pawar
