color mode: It is used to give color in different way. Youtube Video Link...

1.) color name: We can use direct color name such as red, green, orange etc.

2.) hexadecimal (#rrggbb) It is 6 digit hexadecimal code for different color such as "#ff00cc", "#ccccff" etc. we put "#" before 6 digit hexadecimal code.

3.) rgb (rgb(0,255,0)):stands for red, green, blue in this we give red, green and blue color respectively value with min 0 and max 255.

4.) rgba (rgba(0,255,0,0.5)):stands for red, green, blue and alpha. rgba is similar to the rgb and the last value alpha is for opacity its range from 0 to 1.

5.) hsl (hsl(270,60%,60%)): stands for hue, saturation and lightness. hue(0 to 360), saturation(0% to 100%) and lightness(0% to 100%).

6.) hsla (hsla(270,60%,70%,0.5)): stands for hue, saturation, lightness and alpha. hsla is similar to the hsl and the last value alpha is for opacity its range from 0 to 1.
