Print media

There are two types of media:  1. screen (default value of media, used for define css according to the screen)  2. print (used to define css for web page when we print any web page.) Css for print media can be given at 3 places:  1.link rel="stylesheet" media="print" href="print.css"

In this way the css for print is written in external file and it is linked to the webpage using link tag. Here the attribute "media="print"" is necessary to tell the browser that it is as file for print media.

 2.@import "print.css" print; or @import url(print.css) print;

In this way, the print.css file is linked another css file and that file is linked in the web page.

 3.@media print{ css code }

In this way, we can give css for print in the style tag within the same html page.

@page

@page media rule is used to set margin and size of paper used in print media. In this media rule we can use only two properties:  1) margin  2) size Syntax:

@page  {   margin:2in;   size:A3;  }

 Properties:  1.) margin:1in 2cm 0 1cm;

used to set margin of the page during print, we can also give margin from top, right, bottom or left in same way as we give normal margin, we can't use unit in pixel, it can be given in "in"(for inch), "cm"(for centimeter), "mm"(for millimeter), "%"(for percentage). we can also set different margin for different pages. For this 3 types of pseudo classes are used:  1.):first ( to set the margin of first page i.e. page no. 1)  2.):left (to set the margin of all even pages, i.e. left side pages.)  3.):right (to set the margin of all odd pages, i.e. right side pages.) Syntax:

@page:first  {   margin:2in;   size:A3;  }

 2.) size:A3;

used to give the size of paper for print.Following are the pre-defined values of the size property:  1.) Landscape  2.) Portrait (no need to give since the page by default is already in protrait mode.)  3.) auto (page size set to the size of printers default size set by user)  4.) A4, A3, A5, B5, B4 JIS-B5, JIS-B4, letter, legal, ledger etc (are some common size of papers)  5.) We can also give our own size for example: size:8in 16in; (here first value is for width and the second value is for height of the page.) Syntax: 1.) size:landscape; 2.) size:A3 landscape; 3.) size:8in 16in; we can also set different size for different pages. For this 3 types of pseudo classes are used:  1.):first ( to set the size of first page i.e. page no. 1)  2.):left (to set the size of all even pages, i.e. left side pages.)  3.):right (to set the size of all odd pages, i.e. right side pages.) Syntax:

@page:first  {   margin:2in;   size:A3;  }
