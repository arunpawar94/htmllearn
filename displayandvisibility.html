display: display property is used when we want to use property of any tag in another tag.For example if we want padding, margin etc which are used in div tag but they not work in list tag so we use property "display:block;" in list tag.
Syntax:display:block;

Different values are:
  1. none(to hide)
  2. inline(we can use it to show "li" list in one line)
  3. block(to use property of div)

visibility:it is used to hide the content.
Syntax:visibility:hidden;
Youtube Video Link...
Different values are:
  1. visible(default value does not show any effect)
  2. hidden(hide the content, it is similar to the "display:none;" but in "display:none;" the place is occupied by the data to the next of it whereas in "visibility:hidden;" the data next of it not occupied that place )
  3. collapse
