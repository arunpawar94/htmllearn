Properties:

1.) Background-image: to add image in background.
Syntax:background-image:url(images\481336.jpg)
For multiple imagesbackground-image:url(hsjhdjkhjk.jpg),url(jhhsdkhfjkh.jpg);

2.) Background-repeat: it used how the image repeat in background.
Syntax:background-repeat:no-repeat;

values are:
  • repeat(default)
  • no-repeat(image not repeat)
  • repeat-x(repeat but only horizontally)
  • repeat-y(repeat but only vertically)
  • space(there is some space in repeated image)
  • round(image is repeat and stretched)


3.) Background-position: defines the position of background image where to place it.
Syntax:background-position:center center;

values are:
  • left top(default)
  • left center
  • left bottom
  • right top
  • right center
  • right bottom
  • center top
  • center center
  • center bottom
  • we can also give it in percentage(%) such as "20% 30%" here 20% is along x-axis and 30% is along y-axis.
  • we can also give it in pixel(px) such as "200px 100px" here 200px is along x-axis and 100px is along y-axis.

4.) Background: it is shorthand of the above 3 properties i.e. background-image background-repeat and background-position respectively.
Syntax:background:url(images/hjkdhsjhd,jpg) no-repeat right center;

5.) Background-attachment: normally background image scroll with the contain but background-attachment make it fixed or scrollabel.
Syntax:background-attachment:fixed;

There are only 2 values:
  • scroll(default)
  • fixed


6.) Background-size: used to resize the background image.
Syntax:background-size:600px 500px;

The values are:
  • auto(default)
  • cover(cover whole place of the background and crop the image)
  • contain(image is fully fitted in the background and some place may or may not be left.)
  • can be given in pixel(px) "600px 400px" the 600px is for horizontal length and 400px is for vertical length.
  • can be given in percentage(%) "30% 40%" the 30% is for horizontal length and 40% is for vertical length.


7.) Background-origin: define to from where the background-image should start.
Syntax:background-origin:content-box;

There are 3 values:
  • padding-box(It is default value.background-image start along with padding start)
  • content-box(background-image start along with the starting of the content)
  • border-box(background-image start along with starting of the border)


8.) Background-clip: define from where the background color starts.
Syntax:background-clip:content-box;

There are 4 values:
  • padding-box(It is default value.background start along with padding start)
  • content-box(background start along with the starting of the content)
  • border-box(background start along with starting of the border)
  • text(Presently it is not supported by the all browsers we can use prefix for this as shown below.) 
color:transparent;
font-size:70px;
font-weight:900;
background-image:url(images/1736527.jpg);
background-clip:text;
Result of above code: My name is Arun Pawar.
There is image within the text written above.
