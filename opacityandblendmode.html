1.) Opacity:
Youtube Video Link...
Syntax:opacity:0.5;
its value lie b\w 0 to 1. It makes all the content transparent i.e. text,image,background and whatever within the box. and if we want to transparent only background and not the content within it we can use rgba or hsla color mode.
In Internt explorer older version 8 and its below we can use this property by following syntax.
Syntax:filter:alpha(opacity=50);

2.) background-blend-mode
Youtube Video Link...
it is used when we have image and color both in the background, we can give different effects by blend mode.
Syntax:
background-image:linear-gradient(blue,purple), url(images/3977442.jpg);
background-blend-mode:overlay;

Different values are:
  1. normal(default)
  2. color-dodge
  3. hue
  4. multiply
  5. color-burn
  6. color
  7. screen
  8. hard-light
  9. soft-light
  10. saturation
  11. overlay
  12. luminosity
  13. darken
  14. difference
  15. lighten
  16. exclusion

3.) mix-blend-mode
Youtube Video Link...
it is similar to the background-blend-mode, in background-blend-mode it shows effect on background but in mix-blend-mode we show effect on the content.

Syntax:
img { mix-blend-mode:luminosity; }
it is given in image tag as shown above.
The values of mix-blend-mode and background-blend-mode are same.
