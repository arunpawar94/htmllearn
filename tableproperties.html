Properties of the table:

1. border-spacing: used in table selector to give space b\w border of the td, tr and table.
It can given in 2 ways:
1. border-spacing:10px;(it give space of 10px b\w both columns and rows)
2. border-spacing:10px 20px;(it give space of 10px b\w columns and 20px b\w rows)

2. border-collapse: used in table selector to collapse or separate the border of td, tr and table.
It has 2 values:
1. separate (default value it separate the borders form eachother)
2. collapse (used to collapse the borders of td, tr and table)

3. vertical-align: used in td selector to align the data within the cell.
It has three values:
1. top
2. middle (default)
3. bottom

4. padding:used in td and th selector.

5. caption-side: used in caption selector for place caption tag of html on top or bottom of the table.
It has only 2 values:
1. top (default value place the caption above the table)
2. bottom (place the caption below the table)

6. empty-cells: used in table selector to show the cell or hide it only if the cell is empty.
it has only 2 values:
1. show (default value it show the empty cell)
2. hide (hide the if cell is empty)

7. table-layout: used in table selector. In table if the data is increased than its width will increase to the width which we have given. To make it fixed to the width we have given we use table-layout.
It has only 2 values:
1. auto (default value width change if data is increased)
2. fixed (make the width of the table fix that we have given and not increased its width if data is increased)
