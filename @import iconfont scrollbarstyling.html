Scrollbar Styling: They are pseudo elements which are used for styling of the scrollbar.
Youtube Video Link....
These pseudo elements are not supported on browsers so we use prefix for use them.
    There are 4 pseudo elements that are used to styling the scrollbar.
  1. ::scrollbar
    2. it is used to define the width of the scrollbar and on which scrollbar-track lie.
  2. ::scrollbar-thumb
    3. It is used for styling the bar that we move.
  3. ::scrollbar-track
    4. It is used for styling the track on which the bar move.
  4. ::scrollbar-corner
    5. It is used to fill the background color for the space left b/w the vertical and horizontal scrollbar.

for example, we design the body scrollbar on right side of these page.
The syntax we use for designing it is: 
        body::-webkit-scrollbar
        {
            width:28px;
            background-color: #e30f75;
        }
        body::-webkit-scrollbar-track
        {
            border-radius: 10px;
            background-color:darkslategray;
        }
        body::-webkit-scrollbar-thumb
        {
            border-radius: 10px;
            background:radial-gradient(#163d8f,#47afe5,#790487);
        }
