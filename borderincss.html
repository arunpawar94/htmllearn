Using border tag in css


There are 10 types of border style:
1. solid
2. groove
3. dotted
4. dashed
5. ridge
6. outset
7. double
8. none
9. mix
10.inset

css border shorthand:

In border shorthand all border tags used in one tag.
for example:
border-width:5px;
border-style:groove;
border-color:blue;
Can be written as border:5px groove blue;

Mix Style used as:

border-style:ridge dotted dashed double;


here ridge is top,dotted is right, dashed is bottom and double is left border
We can also give only single border top or right or bottom or left only
border-top:3px double red;

Single border only for top
