aspect-ratio:1/2;

This property is used to define ratio b/t width and height of any element.  Syntax:   1. aspect-ratio:1/2; (first value is for width and second value is for height.)   2. aspect-ratio:1; (this way is used when value of width and height is same.)  Important points:
  • From "width" and "height" only one is used with "aspect-ratio" otherwise "aspect-ratio" will become useless.
  • If "width" and "aspect-ratio" then "aspect-ratio" work as minimum height.
  • it is mainly used for make any div, image, or video responsive.
 Example:
on div width:30%; aspect-ratio:1/1.1;
on image width:30%; aspect-ratio:1/1.1; object-fit:cover; object-position:left;
on video width:50%; aspect-ratio:16/9;
Resulted div
Resize the above divs to see effect.

@media (display-mode:fullscreen)

used to change css property when we switch display mode of our browser.  Syntax:
	@media (display-mode:value)
		{
			css property
		}
  We have 3 values we can use in place of value.
  1. browser (default mode of the browser)
  2. minimal-ui (it appears when we save our )
  3. fullscreen (it appears when we press f11 key then browser enters in fullscreen display mode.)
