@supprts:

it checks the condition wheather it is supported by any browser or not on that browser. Syntax:

                @supports (display:grid)
                {
                    body
                    {
                        background-color:red;
                    }
                }
In above synatax,if the browser on which this css run and if it supports the propert "display:grid;" then its background color will change to red.
There are 3 operators, we can use with it that are:

1. not 2. and 3. or

way to use the above operators:

1. not 

            @supports not (display:grid)
            {
                css property
            }
2. and 
            @supports (display:flex) and (display:-webkit-flex)
            {
                css property
            }
3. or 
            @supports (display:flex) or (display:-webkit-flex)
            {
                css property
            }

we can also give multiple condition using above operators.

:placeholder-shown pseudo class

It is a pseudo class. work with only 2 tags i.e. first is "input" and second one is "textarea". it does not work with type time, date, datetime-local, month, week etc. this is used to define css property until the placeholder is shown in input tag. Syntax:

            input:placeholder-shown
            {
                css property
            }
we can also use it with a specific type of input tag. 
            input[type="email"]:placeholder-shown
            {
                css property
            }

scroll-behavior:smooth;

it makes the scrolling of any intarnal page link smooth. There are 2 values of the scroll-behavior. 1. auto (default value does not show any effect) 2. smooth (make the scrolling smooth) there are two way to use it: 1. to use it in whole page we have to use html as a selector. html { scroll-behavior:smooth; } 2. to use particularly in any div, we have to use id or class as a selector. id_name or class_name { scroll-behavior:smooth; } click on the link "home" or "about us" or "contact" to see effect.
