flexbox

Introduction:

1. New module in css3 to easy align elements in different directions and orders. 2. Flexbox is to gives container the ability to expand and shrink elements to best use all the available space. 3. CSS Flexbox layout replaces float layout. 4. New way to build one-dimensional layouts.
Layout and terminology of flexbox: we use property "display:flex;" to make any div flexbox.
Properties used in flexbox are as follows:

  1. flex-direction
  2. flex-wrap
  3. flex-flow
  4. flex-grow
  5. flex-shrink
  6. flex-basis
  7. flex
  8. justify-content
  9. align-cotent
  10. align-items
  11. align-self
  12. order

flex-wrap and flex-flow

a.) flex-wrap These propery is used when flex item overflow the flex container. Syntax:flex-wrap:wrap; These property has following 3 values: 1. nowrap (default value, It does not show any effect) 2. wrap (make the flex item to be in flex container) 3. wrap-reverse (make the flex item to be in flex container but also reverse the main axis) b.) flex-flow These property is the shorthand of the two properties i.e. "flex-direction" and "flex-wrap". Syntax:flex-flow:column wrap; (here first value is for property "flex-direction" and second is for "flex-wrap" resp.)

order:0;

These property is given in flex item. It is used for positioning any flex item in flex container. Its value is the numerical value which can be negative or positive. "0" is the default value. Syntax:order:5;


flex-grow:1;

These property is given in flex item. It is used when we have space remains left in flex container to give that space to the flex item. its value is numerical value. "0" is the default value. Syntax:flex-grow:1;

flex-basis:100px;

These property is given in flex item. It is used to give width to the flex item. It act like a maximum width. Its value can be given in pixel and percentage. "auto" is the default value. Syntax:flex-basis:200px;


flex-shrink and flex

a.) flex-shrink:0; It is given in flex item.These makes the flex item wheather it change its with or not. Its value is given in positive numerical value. > "1" is the default value. it makes the flex item shrinkable. > "0" it makes the width of the flex item fixed so that it will not shrink. > any other value rather than "1" or "0" for example "3" then the flex item will shrink 3 times than its normal width. Syntax:flex-shrink:0; b.) flex:0 0 200px; It is the shorthand of 3 properties "flex-grow", "flex-basis" and "flex-shrink" respectively. Syntax:flex:0 0 150px;


flexbox with "margin:auto;"

used to give margin in flex item. we can also use 4 properties of margin. 1. margin-left:auto; 2. margin-right:auto; 3. margnin-top:auto; 4. margin-bottom:auto;

if any flex item is given with "display:flex" then it is called nested flex container, and we can use all flexbox properties here.
