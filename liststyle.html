list-style: use in designing ordered or unordered list: Youtube Video Link...

There are following 4 properties:
  1. list-style-type
  2. list-styel-position
  3. list-style-image
  4. list-style (it is the shorthand for the above 3 properties.)


1.) list-style-type:circle;
   a.) values in ul are:disc(default),circle,square,none.
   b.) values in ol are:decimal(default),upper-alpha,lower-alpha,upper-roman,lower-roman,none.

2.) list-style-position:inside;
   There are only 2 values:outside(default) and inside.

3.) list-style-image:url(icons/rightarrow.png);
by these we can add image in place of disc,circle,square and so on in list as shown below.
  • Apple
  • Orange
  • Mango
  • banana


4.) list-style:none inside url(icons/rightarrow.png); It is the shorthand of above 3 properties.(type position image)
