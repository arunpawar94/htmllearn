list-style: use in designing ordered or unordered list: Youtube Video Link...

There are following 4 properties:
  1. list-style-type
  2. list-styel-position
  3. list-style-image
  4. list-style (it is the shorthand for the above 3 properties.)


1.) list-style-type:circle;
   a.) values in ul are:disc(default),circle,square,none.
   b.) values in ol are:decimal(default),upper-alpha,lower-alpha,upper-roman,lower-roman,none.

list-style-type:"string";
we can also give string in the list-style-type which works for both order list and unordered list. It is given in single or double quotes.
some example are:
1. list-style-type:"aurn-";resulted output:
  1. Apple
  2. Orange
  3. Mango
  4. Banana

we can also use any symbol. just search "html symbol" online copy that symbol from any website and paste at the place of string. As shown below. the html symbol is taken from website "www.toptal.com"
2. list-style-type:"☃ ";resulted output:
  1. Apple
  2. Orange
  3. Mango
  4. Banana

3. list-style-type:"⚰ ";resulted output:
  1. Apple
  2. Orange
  3. Mango
  4. Banana

4. list-style-type:"☘ ";resulted output:
  1. Apple
  2. Orange
  3. Mango
  4. Banana


2.) list-style-position:inside;
   There are only 2 values:outside(default) and inside.

3.) list-style-image:url(icons/rightarrow.png);
by these we can add image in place of disc,circle,square and so on in list as shown below.
  • Apple
  • Orange
  • Mango
  • banana


4.) list-style:none inside url(icons/rightarrow.png); It is the shorthand of above 3 properties.(type position image)
