box-shadow: It is used to give shadow to the box. youtube video link
It can be written as: "box-shadow: 6px 6px 3px 10px red;" (here the values are h-offset, v-offset, blur, spread, color resp.)
h-offset is for horizontal value(it can be negative);
v-offset is for vertical value(it can be negative)
blur is used to blur the shadow.(it is optional)
spread is used to spread the shadow.(it is optional)
