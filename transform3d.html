transform 3D: It is similar to 2D transform but here we have z-axis also. Youtube Video Link....
following 12 value of the transform 3D:

1. none: (default value,no transform)

2. rotatex(angle): (rotating x-axis,value is in degree, it can be negative or positive both)
Syntax:transform:perspective(800px) rotatex(70deg);
we use "perspective(800px)" value here so that we can see effect clearly.
Result:


3. rotatey(angle): (rotating y-axis,value is in degree, it can be negative or positive both)
Syntax:transform:perspective(800px) rotatey(70deg);
we use "perspective(800px)" value here so that we can see effect clearly.
Result:


4. rotatez(angle): (rotating z-axis, it looks similar to rotate in 2D transform,value is in degree, it can be negative or positive both)
Syntax:transform:perspective(800px) rotatez(70deg);
Result:


5. translatez(200px): (we are moving on z-axis, value is in pixel, it can be negative or positive both)
Syntax:transform:perspective(800px) translatez(200px);
we use "perspective(800px)" value here so that we can see effect clearly.
Result:


6. translate3d(-80px,80px,200px): (we are moving on x-axis,y-axis and z-axis in same order, value is in pixel, it can be negative or positive both)
Syntax:transform:perspective(800px) translate3d(-80px,80px,200px);
we use "perspective(800px)" value here so that we can see effect clearly.
Result:


7. scalez(4): (we are changing size of the element on wrt to z-axis,here we use "scalez(4)" means value 4 times the size of normal element, "1" is the default value,it can be negative or positive both)
Syntax:transform:perspective(800px) scalez(4) rotatex(40deg);
we use "perspective(800px)" and "rotatex(40deg)" value here so that we can see effect clearly.
Result:


8. scale3d(2,2,2): (we are changing size of the element on wrt to x-axis, y-axis and z-axis,here we use "scalez(2,2,2)" means value 2 times the size of normal element,it can be negative or positive both)
Syntax:transform:perspective(800px) scalez(4) rotatex(40deg);
we use "perspective(800px)" and "rotatex(40deg)" value here so that we can see effect clearly.
Result:


9. matrix3d(0.6, 0.1, 0.7, 0, -0.5, 0.8, 0.1, 0, -0.6, -0.5, 0.5, 0, 0, 0, 0, 1): (all value of 3D transform can be given in matrix3d)
Syntax:transform:matrix3d(0.6, 0.1, 0.7, 0, -0.5, 0.8, 0.1, 0, -0.6, -0.5, 0.5, 0, 0, 0, 0, 1);
we use "perspective(800px)" and "rotatex(40deg)" value here so that we can see effect clearly.
