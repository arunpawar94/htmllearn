text-overflow helps when text overflow the box. By this we can show that there is text in continue. It comes in css3 version. Mainly use when we use "white-space:nowrap;"
Youtube Video Link

There are 3 values:
  • clip (default)
  • ellipsis
  • string (these value does not work in every browser. It works in mozilla firefox)
without text-overflow
width:400px;
border:6px groove purple;
background-color:#ccccff;
white-space:nowrap;
overflow:hidden;
