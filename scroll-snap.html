scroll-snap

scroll-snap is used when we are scrolling between different section as in scroll-behavior and we want to that it automatically jump on that section whichever have larger part. There are two properties used for scroll-snap:  1. scroll-snap-type:y mandatory; It is given in parent element. It defines the direction of scrolling.    There are 5 values for scroll-snap-type:
  1. y mandatory (used for vertical scrolling. scrolling always happen on larger part)
  2. x mandatory (used for horizontal scrolling. scrolling always happen on larger part)
  3. both mandatory (used for vertical as well as horizontal scrolling. scrolling always happen on larger part)
  4. y proximity (used for vertical scrolling. scrolling not necessory to happen. it scrolls only when there is larger part is display of any section i.e. about 70 to 80 or above it.)
  5. x proximity (used for horizontal scrolling. scrolling not necessory to happen. it scrolls only when there is larger part is display of any section i.e. about 70 to 80 or above it)
 2. scroll-snap-align:start; it is given on child element, which scroll. It align the section element which we want to scroll.
   There are 3 values for scroll-snap-type:
  1. start. (it align the element from start)
  2. center. (it align the element from center)
  3. end. (it align the element from end)

  1. blur(4px) value is given in pixel(px).
  2. brightness(60%) value is given in percentage(%). 100% is normal value.
  3. contrast(40%) value is given in percentage(%). 100% is normal value.
  4. drop-shadow()
  5. grayscale(50%) value is given in percentage(%).
  6. hue-rotate(120deg) value is given in degree(deg). we can given its value from 0deg to
  7. invert(60%) value is given in percentage(%).
  8. opacity()
  9. sepia(90%) value is given in percentage(%).

  11. none It is default value. it does not show any effect.

display:table;

Using this property we can use any element such as div as table tags. We can use div tag to make any table. On this div we can use the properties that we use in the table tag. Following value of display used to make different table tag.
  1. display:table; (it make the div as "table" tag in html. All the below display values are used on the div within this div.)
  2. display:table-cell; (it make the div as "td" tag in html. for making cell in table)
  3. display:table-row; (it make the div as "tr" tag in html. for making row in table.)
  4. display:table-caption; (it make the div as "caption" tag in html. for caption of the table)
  5. display:table-header-group; (it make the div as "thead" tag in html. for header row of the table)
  6. display:table-footer-group; (it make the div as "tfoot" tag in html. for footer row of the table.)
  7. display:table-row-group; (this value is used for making group of rows. And we put those div with property "display:table-row;" that we want to make group within the div of property "display:table-row-group;". )
  8. display:table-column-group; (this value is used for making group of columns. It is difine on the div within div with property "display:table;". First we have to make some div with property "display:table-column;" within this div with property "display:table-column-group;". Each div with property "display:table-column;" represent all the cell within that column. the div with property "display:table-column;" as many as the no. of columns and not more than that.)
  9. display:table-column; (each div with this property represent each column. this div define within the div with property "display:table-column-group;".)

accent-color:red;

In input tag we have type "checkbox" and "radio", which have default color blue, to change its color we can use property "accent-color:pink;". Syntax: 
input
    {
      accent-color:pink;
    }
