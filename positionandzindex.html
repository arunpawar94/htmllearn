position:It is used to give location of the div. Youtube Video Link...
It necessory to use following properties with "position".
1.)left(for x axis)
2.)right(for x axis)
3.)top(for y axis)
4.)bottom(for y axis)
from left or right we have to use only one also from top and bottom we have to use only one.

Syntax:
position:sticky;
left:-4px;
bottom:4px;

The values of position properties are:
  1. static
  2. relative
  3. absolute
  4. fixed
  5. sticky


2.)relative: In relative the position of the div is with respect to its original position and scroll with the page.

3.)absolute: In absolute the position of the div is with respect to absolute screen position and scroll with the page.

4.)fixed: In fixed the position of the div is with respect to absolute screen position and does not scroll with the page.

5.)sticky: In sticky the position of the div is with respect to absolute screen position and scroll with the page but after reaching at the given position it does not scroll with the page.
