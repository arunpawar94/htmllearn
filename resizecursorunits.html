resizeit used to resize the size of any div or anything else.It is necessory to use "overflow:auto;" with resize.
Youtube Video Link....
There are 4 values:
1. none (default value. It can be used in "textarea" tag, since "textarea" is resizable by default to make it non-resizable we use this value.)
2. horizontal (it helps only resize div horizontally)
3. vertical (it helps only resize div vertically)
4. both (it helps resize div vertically as well as horizontally)
Example: 
	overflow:auto;
	resize:both;

units: Youtube Video Link....
Units are devided mainly in 2 catogories:

1. Absolute: There values are fixed. They does not depend on other values.
  1. cm (centimeters)
  2. mm (millimeters)
  3. in (inches)
  4. px (pixels)
  5. pt (points)
  6. pc (picas)

2. relative: whose value change or depend on parent div or anything else such as body or html.
  1. em (change size according to the parent.For example if body has 10px text and if we use 0.5em than it means 50% of body text which is 5px)
  2. rem (relative em. value depends on root tag such as html if browser text is set to 16px than 2rem means 200% of it which is 32px and does not depend on its parent tag)
  3. ch
  4. ex
  5. vw (viewport width,50vw means 50% of the viewport horizontally)
  6. vh (viewport height,110vh means 50% of the viewport vertically)
  7. vmax (viewport maximum, it takes the value w.r.t which is greater b\w viewport height or viewport width, for example width of screen is 900px and height of screen is 1000px and if we give 60vmax than it equal to 60% of greater value which is height of the screen )
  8. vmin (viewport minimum, it takes the value w.r.t which is smaller b\w viewport height or viewport width, for example width of screen is 900px and height of screen is 1000px and if we give 60vmin than it equal to 60% of smaller value which is width of the screen)
  9. % (percentage)
