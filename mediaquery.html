Media type:
There are 4 types of media:
1.) print (helps in change when we want to print the web page)
2.) screen (helps in change when it runs different size of the screen such as on mobile, tablet etc.)
3.) speech (helps in making web page readable)
4.) all (it covers all above 3 types)
