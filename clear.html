clear: When we use float in any div then this div overlap the another div. To avoid this situation we use clear property in css.
Youtube video link
It can be written as:
clear:left;
There are 3 values of clear property:
1.left ( use if in previous div used float:left then left side of not to next div overlap )
2.right ( use if in previous div used float:right then right side not to next div overlap )
3.both ( it is used to avoid overlap of next div wheathere we use left or right float used in previous div)
