with using word-wrap
It has only 2 values:

a) normal (default value. it does not show any effect.)
word-wrap:normal;

b) break-word
word-wrap:break-word;

B)word-break:It is used if in a line there is a last word which is not fitting in the div then it comes in next line and leave some space in previous line as shown below.

without word-wrap
Banned items/electronic gadgets such as Mobile phones, pager, watches, Bluetoothenabled.

with word-wrap
It has only 2 values:

a) keep-all (default value. it does not show any effect.)
word-break:keep-all;
Banned items/electronic gadgets such as Mobile phones, pager, watches, Bluetoothenabled.


b) break-all (in this all letters of every word is separated from eachother)
word-break:break-all;
