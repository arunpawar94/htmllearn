variable and var() function:for when we use the same value like color at different places and if we want to change the color then we have to change that color on each and every places which is also a time comsuming. To overcome this problem we use variable.
Youtube Video Link....
syntax: 
		:root
		{
		  --variableName:blue;
		}
		body
		{
		  background-color:var(--variableName);
		}
in ":root" we define the variable of "variableName" store the value of blue color. and in body selector, we change the background-color by calling a var() function with variableName. In ":root" the scope of variable is global and if we define in any class or id its scope will be local.

clip-path: it is used to crop the image or div in specific shape. Youtube Video Link....
Syntax:clip-path:circle(33% at 35% 55%);
Example:
Image with clip-path
Normal Image

There are 4 values of clip-path are as follow:
1. circle(33% at 35% 55%) (there are 3 units we have to given 33% is the radius and 35% is for x axis 55% is for y axis from left and top resp. )
2. ellipse(20% 40% at 35% 55%) (there are 4 units we have to given 20% is the horizontal radius, 40% is the vertical radius and 35% is for x axis 55% is for y axis from left and top resp. )
3. inset(20% 30% 40% 50%) (there are 4 units we have to given 20% is from top, 30% is from right, 40% is from bottom and 50% is from left of the image )
4. poygon(20% 30%,40% 50%,60% 70%) (In this we can give as many point as we want. we have to give cordinates of every point separated by commas. here we have 3 points.)

from website https://bennettfeely.com/clippy/ we can copy the css code for different shapes.

filter filter property is used to change the appearance of the images Youtube Video Link....
Syntax:filter:
different values of the filter are:
1. none (default value remove all effect)
2. blur(5px);
       (its default value is 0px)
3. brightness();
        (its value is given in %, 100% is its default value, at 0 it become black, above 100% it become brighter )
4. contrast();
        (it is given in %, 100% is default value, its value can be 0 and above 100%)
5. drop-shadow();
        (it is similar to box-shadow)
6. grayscale();
        (0 is default value,min is 0% and max is 100%)
7. hue-rotate();
       (its value is given in "deg" 0deg in min and 360deg is max value)
8. invert();
        (0% is default value, 0% is min and 100% is max)
9. opacity();
        (100% is default value,0% is full transparent and 100% is fully non transparent)
10. saturate();
        (100% is default value, we can also give its value above or below 100%)
11. sepia();
        (0% is default value, 0% is min and 100% is max)
12. url() ()


We can also give multiple filters in following way:
filter:blur(2px) hue-rotate(200deg) saturate(200%);
