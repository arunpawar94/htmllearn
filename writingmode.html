text-orientation It is used with "writing-mode" property. used to rotate the letter within word.

There are 4 values:
  1. mixed(default, It does not show any effect.)
  2. upright
  3. sideways(currently not supported by any browser)
  4. sideways-right(currently not supported by any browser)
example:
without text-orientation
writing-mode:vertical-rl;
Arun Pawar

using text-orientation
writing-mode:vertical-rl;
text-orientation:upright;
Arun Pawar
