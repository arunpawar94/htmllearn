attr() function: This function is used to show value of attribute name given in attr(attribute_name) function. It is always used with "content" property.
Youtube Video Link...
Syntax: 
tag_name::after
            {
                content:attr(class);
            }

in above syntax, tag_name is the name of the tag where we want to use attr() and "class" is the attribute name we use within attr() function to show the value of class attribute. We can also use another attribute name in place of class.

We can also make our own attribute using "data-" within tag. for example: 
data-attribute_name="value"


counter-increment used to increase the value
Youtube Video Link...
It has 3 properties, all are used simultaneously:
1. counter-reset:counter_name 0; (used to give counter name and starting value)
2. counter(counter_name); (It is a function to call the counter we define with counter name)
3. counter-increment:counter_name 1; (to increase the value of the counter)
Syntax: 
            body
		{
			counter-reset:counter_name 0;
		}
			
		h2::before
		{
			content:"chapter:" counter(counter_name);
			counter-increment:counter_name 1; 
		}

We can also use roman or alpha value in place of alpha-numeric value. 
                body
            {
                counter-reset:counter_name;
            }
                
            h2::before
            {
                content:"chapter:" counter(counter_name, upper-alpha);
                counter-increment:counter_name 1; 
            }
There are 4 value we can use i.e. upper-alpha, lower-alpha, upper-roman or lower-roman.
