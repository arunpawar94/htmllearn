transform-style: It is given in parent element.by this we are able to see back of the element. It is used with transform property. It gives it 3d effect. Youtube Video Link....

There are 2 values;
1. flat: (It is default value)
Syntax:transform-style:flat;

1. preserve-3d: (it gives 3d look, by we are able to see back of the parent element)
Syntax:transform-style:preserve-3d;
in above div,we give parent div rotatey and child div rotatex so that we can see the effect of transform-style.


backface-visibility it helps in hidden or visible the backface of the element, it only works with transform property. Youtube Video Link....

There are 2 values;

1. visible: (It is default value)
Syntax:backface-visibility:visible;
here we have 2 div one is front and another is back. we rotate the back div at y-axis at 180deg. Since the backface of the back div is visible so we are not able to see the front div which is behind of it.
