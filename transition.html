transition: make the change in effect smooth. Youtube Video Link....
Different properties are as follows:
1. transition-property
2. transition-duration
3. transition-delay
4. transition-timing-function
5. transition

transition-property:it apply on which property of css we want to apply transition such as width, background, transform etc.
Syntax:transition-property:width,height,background;

transition-duration: it used in how long in time a transition should be complete in. It can be in "s" or "ms".
Syntax:transition-duration:2s 3s 4s;(multiple time for diferent properties in transition-property we have give such as 2s for width,3s for height and 4s for background)

transition-timing-function: if we want the transition slow or fast.
Syntax:transition-timing-function:ease-in-out(slow in starting and normal end)
Unit Description
ease slow start,then fast,then end slowly
linear same speed from start to end
ease-in slow start
ease-out slow end
ease-in-out slow start and end
step-start
step-end
steps(4,end) we can give in how many steps transition completed
cubic-bezier(2s,3s,4s,5s) duration of each step, 4 is fixed number of steps can given here


transition-delay: used to how long it take time before transition start.
Syntax:transition-delay:2s; (transition start after 2s)


transition: it is shorthand of above 4 properties.
Syntax:transition:all 2s,3s,4s linear 1s;(property, duration, timing-function, delay respectively it should be in the same order as written here)

Example: 
#id1
{ width:600px;
  height:350px;
  color:blue;
  background-color:yellow;
  transition-property:color,width,background-color;
  transition-duration:2s;
}
#id1:hover
{ width:500px;
  color:yellow;
  height:450px;
  background-color:blue;
}
