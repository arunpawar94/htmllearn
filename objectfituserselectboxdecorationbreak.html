object-fit: It is the new way to make responsive the images or videos.Youtube Video Link....
Syntax: object-fit:contain;

There are following 5 values:
1. fill (here image cover the whole space it stretch or screw itself if required to fill the remaining space)
2. contain (image resize itself, according to the space and image does not crop always show the whole image)
3. cover (cover the given area and does not stretch or screw it crops according to the space available if we resize the browser)
4. scale-down (it is similar to the contain value)
5. none (cover the given area and does not stretch or screw it crops according to the space available if we resize the browser)

object-position It is another property which is used with object-fit to give position of the reference point w.r.t which the image resize.

Syntax: object-position:top;

There are following 5 values:
1. center (default value)
2. top
3. right
4. bottom
5. left
we can also give customize value such as "object-position:30px 50px;"


user-select: It restricts the user from copying the data from website.Youtube Video Link....
Syntax: user-select:none;

There are following 4 values:
1. auto (default value, does not restrict the user from copying the data)
2. none (restrict the user from selecting the data and copying it)
3. text (text is similar to auto but in auto if browser setting not allowed to copy then in auto also data select but in text it always allow to select the data)
4. all (select the whole text in single click)
