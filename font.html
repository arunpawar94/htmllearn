font: It is used to define text in different styles. Youtube video link...
Different properties of the font are:
1.font-style:italic;
used to make text italic.
There are 3 values:
a) normal (default value)
Eg:My name is Arun Pawar.
b) italic (used to make text italic, but only those text which has italic version of that text format)
Eg:My name is Arun Pawar.
c) oblique (used to make text italic wheathere it has italic version of that text format or not)
Eg:My name is Arun Pawar.

2.font-variant:small-caps;
used to make text capital. And the starting letter of the sentence is slightly have more length than the other letter of the sentence.
There are only 2 value:
a) normal (default value)
Eg:My name is Arun Pawar.
b) small-caps (make the letter capital)
Eg:My name is Arun Pawar.

3.font-weight:700;
Used to make the text bold.
The values are;
a) normal (default value)
Eg:My name is Arun Pawar.
b) bold
Eg:My name is Arun Pawar.
c) lighter (the lighter value make the text slightly thin than the normal text i.e. its parent value)
Eg:My name is Arun Pawar.
d) bolder (the bolder value make the text slightly thick than the normal text i.e. its parent value)
Eg:My name is Arun Pawar.
e) 100 to 900 (we can also give it in numerical value starting from 100 to 900. The 700 is the equal to bold value)
Eg:My name is Arun Pawar. (font-weight:900;)
4.font-size:25px;
Used to define the size of the text.
The value of font-size can be given in following way:
a) In pixel(px)
Eg:My name is Arun Pawar. (font-size:25px)
b) In em
It adjust the font according to its parent text property.
If parent div have 18px font size than 0.5em means 9px;
If parent div have 18px font size than 1em means 18px
If parent div have 18px font size than 2em means 36px
Eg:My name is Arun Pawar. (font-size:2em)

c) In percentage(%)
It adjust the font according to its parent text property.
If parent div have 18px font size than 50% means 9px;
If parent div have 18px font size than 100 means 18px
If parent div have 18px font size than 200% means 36px
Eg:My name is Arun Pawar. (font-size:200%)

5.line-height:10px;
used to give space btn lines. It can be given in "em" or "px"(pixel).

6.font-family:arial;
It is used for different types of font style such as arial, verdana, calibri etc.
code written as:
1.font-family:Lucida Handwriting;
Eg:My name is Arun Pawar.
1.font-family:cooper black,arial,Lucida Handwriting;(In this we give multiple font-family if any browser does not support cooper black than it show text in arial, if arial is also not support than it show text in Lucida Handwriting and so on.)
Eg:My name is Arun Pawar.
