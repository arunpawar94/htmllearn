transform 2D: In 2d transform we can rotate,scale,skew any element in cssYoutube Video Link....
following 12 value of the transform 2D:

1. none: (default value,no transform)

2. rotate(angle): (rotate the element,value is in degree, it can be negative or positive both)
Syntax:transform:rotate(270deg);
Result:


3. skewx(angle) (rotate the wrt element x-axis,value is in degree, it can be negative or positive both)
Syntax:transform:skewx(20deg);
Result:


4. skewy(angle) (rotate the wrt element y-axis,value is in degree, it can be negative or positive both)
Syntax:transform:skewy(20deg);
Result:


5. skew(angle,angle) (rotate the element wrt x-axis and y-axis,value is in degree, it can be negative or positive both)
Syntax:transform:skewy(20deg);
Result:


6. translatex(-80px) (it change the position of the element horizontally, it can be negative or positive both)
Syntax:transform:translatex(-50px);
Result:


7. translatey(80px) (it change the position of the element vertically, it can be negative or positive both)
Syntax:transform:translatey(80px);
Result:


8. translate(-80px,80px) (it change the position of the element horizontally and vertically respectively, it can be negative or positive both)
Syntax:transform:translate(-80px,80px);
Result:


9. scalex(1.5) (shrink or expand the element horizontally,1 is default value,its value can be negative or positive)
Syntax:transform:scalex(1.5);
Result:


10. scaley(1.5) (shrink or expand the element vertically,1 is default value,its value can be negative or positive)
Syntax:transform:scaley(1.5);
Result:


11. scale(1.2) (shrink or expand the element horizontally and vertically,1 is default value,its value can be negative or positive)
Syntax:
1.transform:scale(1.2);
2. transform:scale(1.2,1.5);
Result:


11. matrix(1.3,0.2,0.3,1.5,-80,80); (by matrix,we can give following value in shortly in following order "matrix(scalex(),skewy(),skewx(),scaley(),translatex(),translatey())" )
Syntax:transform:matrix(1.3,0.2,0.3,1.5,-80,80);
Result:
2D transform


the above all value has transform wrt origin at center. We can also change its center by following property:
transform-origin:left top;
we can give value are as follow:
1. certer center(default)
2. left top
3. right top
4. left bottom
5. right bottom
6. 20% 30% (we can define custom origin according to us 20% for horizontal and 30% verticaly)
